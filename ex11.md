# Exercice 11

Étant donné que ce projet utilise JUnit 5. nous avons 2 choix :
- Modifier la version de JUnit pour utiliser JUnit 4.
- Ajouter la dépendance :
```xml
<dependency>
    <groupId>org.junit.vintage</groupId>
    <artifactId>junit-vintage-engine</artifactId>
    <version>5.10.2</version>
    <scope>test</scope>
</dependency>
```
J'ai décidé de rester avec JUnit 5 et d'ajouter la dépendance.
